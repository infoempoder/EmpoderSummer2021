Hello!

This website is about the transgender community. In this website you will find information about transgender history,gender and pronouns, transphobia, and mental health.

Why I Made This Website

As a Trans Masc myself I felt like people didnt know much about the trans community, so i decided to make a website about it so i can help spread information about different things in the community and hopefully spread more awareness about the community.
